# Screenshot Evidence

This directory is reserved for public screenshot evidence used by the root
`README.md`.

Do not add screenshots that expose:

- AWS access keys or secret keys
- account recovery details
- private emails beyond approved budget/contact evidence
- tokens
- kubeconfigs
- Terraform state
- secret values
- private interview, resume, or job-search material

Required screenshots before publishing:

- Terraform apply success
- AWS EKS cluster and node group
- ECR image repository
- Kubernetes resources from `kubectl`
- application running
- GitHub Actions success
- Helm release output
- Argo CD dashboard
- Prometheus targets
- Grafana dashboard
- Alertmanager
- CloudWatch logs
